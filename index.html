Getting Started

Viewing GMODEL Files

  1. Double-click any .gmodel file
  2. Use the file browser to navigate contents
  3. Click any file to preview it
  4. Use controls for media playback
  5. View file information in the panel

Advanced Configuration

  1. Access the settings panel
  2. Customize appearance and behaviors
  3. Configure playback settings
  4. Set up auto-save preferences

Media Display Mode

  1. Launch via GOONWARE.lnk
  2. Add model files to the models folder
  3. Configure display preferences
  4. Select monitors for display
  5. Click "Start" to begin
